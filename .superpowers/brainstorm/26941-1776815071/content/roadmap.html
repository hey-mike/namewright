12-Month Product Roadmap

Three phases: Make it shippable → Make it findable → Make it a platform.

Phase 1 · Month 1–2 · Foundation
Make it shippable & launchable
Backend / API proxy
Move Anthropic call to Next.js API route. Key stays server-side.
Freemium gate
Show 3 candidates free, blur the rest. "Unlock full report" CTA.
Stripe Checkout
One-time $19 payment. No account required. Session token unlocks report.
PDF export
"Download report" — founders want to share with co-founders.
Affiliate links
Namecheap + Trademark Engine links in results. Low effort revenue.
ToS + Disclaimer
"Not legal advice" language on report. Liability framing.
Launch target: Product Hunt + Show HN + r/startups same week. Goal: 20–50 paying customers, qualitative feedback.
Phase 2 · Month 3–6 · Distribution
Make it findable
SEO content engine
15–20 articles: "how to name a startup", "trademark search guide", "best domain alternatives 2025".
Shareable report links
Public URL for reports. Founders share → backlinks → SEO.
Real WHOIS integration
Live domain availability via WHOIS API. Replace "inferred" with "confirmed".
Email capture
Optional email on results page for "get notified if domains become available".
Testimonial wall
Collect + display from Phase 1 customers. Social proof for conversions.
Reddit / IH community presence
Genuine participation in startup communities. Tool as helpful resource.
Goal: 500+ monthly organic visitors, 25+ reports/month, first affiliate revenue.
Phase 3 · Month 7–12 · Expansion
Make it a platform
Live USPTO integration
Real trademark search via USPTO API. Premium signal, higher confidence scores.
Agency / Pro tier
$49/mo for 10 reports/mo. Targets consultants and accelerators.
Brand brief export
Beyond names: brand voice, competitor landscape, naming rationale doc.
Trademark filing partner
Deep integration with Trademark Engine / Corsearch for in-product filing.
API access
Developers can call your report engine. B2B integrations.
Multi-market TM
EUIPO + IP Australia + UKIPO coverage in one report.
Goal: $2k–5k MRR, first recurring subscription customers, partnership signed.
North Star Metric: Reports generated per week. Everything else (revenue, SEO rank, conversions) is downstream of this number.
